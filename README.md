\# Samurai \& Shinsengumi AI Guide



An interactive English chatbot that helps international users learn about the Shinsengumi, a famous samurai police force from late Edo-period Japan.



This app is built with Streamlit and uses the OpenAI API.  

Users must provide their own OpenAI API key. All API usage is billed to the user's OpenAI account.



---



\## Features



\- Preset question buttons (5W1H + Code of Conduct)

\- Free text question input with character limits to save API cost

\- Short, clear AI answers optimized for low token usage

\- No API keys are stored or logged



---



\## How to Use



1\. Open the app in your browser

2\. Enter your OpenAI API key

3\. Click one of the preset question buttons or ask your own question in English

4\. Read the AI-generated answer

5\. Click "Back to Menu" to ask another question
